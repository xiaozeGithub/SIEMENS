# reactErpTemplate
# 项目介绍

项目使用 react,react-router,reduxbebel,webpack。
app 前端代码
config 基础配置
utils 各个应用(app) 能够公用到的东西(echarts 主题在这里)
webpack 项目构建相关

# how to run
```
    npm install
    npm run dev
```
本地开发启动 localhost:8080


## 项目目录结构

### app 项目前端资源
* components 页面公用组件按照模块来分
* containers 页面部分
* routes 路由部分
* static 静态的css与图片
* utils 一些通用方法和模块
